### State tile grid (session 4 continuation)
Added a 10×5 alphabetical grid of state abbreviation tiles below the tabset panel (always visible on both Map and State Details tabs). Tiles mirror the map fill state and serve as an equal-size alternative to clicking small map polygons (RI, CT, DE, etc.).

**Implementation:**
- `state_abbrs_alpha` vector (50 states, A–Z) + `tile_grid_html` static HTML generated before UI
- Tiles rendered once as static HTML — no `renderUI` re-render on hover/fill changes
- Colors updated via `sendCustomMessage("update_tiles", named_list)` JS handler; `data-fill-color` attribute stored on each tile for `tileHoverOff` reference
- Tile click → `Shiny.setInputValue("tile_click", abb)` → same server flow as map polygon click (select state, switch to State Details tab)
- iframe height increased from 960px to 1100px in `website/map.qmd`; map height reduced from 520px to 500px

### Bidirectional map ↔ tile hover sync
- **Map polygon hover → tile outline**: `observeEvent(input$map_shape_mouseover)` → `sendCustomMessage("set_tile_hover", abb)` → JS toggles `.tile-map-hover` CSS class (amber outline). `map_shape_mouseout` clears it.
- **Tile hover → map polygon border**: Pure JS via `tileHoverOn`/`tileHoverOff` global functions. Uses `HTMLWidgets.getInstance(el).map` to get Leaflet map object, then `eachLayer` to find matching polygon by `layerId` and call `layer.setStyle({color: '#e8a830', weight: 2.5})`. No Shiny roundtrip — instant.
- Behavior is symmetric: both directions produce amber border on the polygon and amber outline on the tile.
- Known edge case: if user hovers tile → drags onto polygon → drags off polygon, `highlightOptions` `resetStyle()` briefly clears the polygon border (reverts to stored original style). Only occurs in that specific gesture; not noticeable in normal use.

### UpSet hover-stuck fix
- `plotly_unhover` is unreliable (doesn't always fire when moving quickly off a bar)

- Server `observeEvent(input$upset_mouseleave)` clears hover state (unless a persistent selection is locked)
